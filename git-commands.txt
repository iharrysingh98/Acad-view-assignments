Basic Git Commands:

To configure Git: $ git config --global user.name "username"
                  $ git config --global user.email "email-id"
  Note:this command is used to configure the author name and email address to be used with our commits.   

To create a local repository we use : git init

To add a file:  git add <filename>

To determine what branch the local repository is on we use : git branch

To list the files you've changed and those you still need to add or commit we use : git status

To Record the changes made to the files to a local repository : git commit -m

To connect a local repository with a remote repository we use : git remote add origin <url >

To Send changes to the master branch of our remote repository: git push origin master

 To Fetch and merge changes on the remote server to our working directory we use:  git pull ,git merge
